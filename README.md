## Modify the project

### Start the project
<br />

To modify the project you need to have node installed on your computer and install its dependencies:

- Install npm
    ```js
    npm install npm@latest -g
    ```
- Install the dependencies
    ```js
    npm install 
    ```
- Start the project 
    ```js
    npm start
    ```
- Relese a build
    ```js
    npm run build
    ```

- JS file with programming (component.js)
